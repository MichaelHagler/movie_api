MyFlix Documentation

Welcome to MyFlix! A server-side application that shows infromation about movies, and gives users the ability to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None Get a list of movies that includes info on title, genre, director, image, and if it's featured
Get data about a single movie by title /movies/:title GET None Same info as the list of movies but just one specified movie
Get data about a genre /movies/genre/:Name GET None Get a genre and a description of that genre
Get data about a single director /movies/director/:Name GET None Get info about the director including name, birth, bio
Add a user /users POST A JSON object user name, and favorite movies. uuid asigned to user A user will have a username, password, email, birthday, and a favorite movie list
Add a movie to user's list /users/:id/:movieTitle POST A JSON object holding data with the movie title A text message stating that the movie has been added to user's list
Get user by username /users/:username GET None Gets user's details(username, password, email, birthday, and a favorite movies list)
update user /users/:username PUT None A JSON oject holding the updated user's info
Remove a movie by title from user's list /users/:id/:movieTitle DELETE None A text message indicating the movie was successfully removed
Remove user /users/:id DELETE None A text message indicating the user has been removed
